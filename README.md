# EncodingChecker
Python script that verify the encoding of all files in a given directory
Automatically install and imports chardet (the lib used for guessing the encoding)

## Output example

```console
File                                          Encoding
sample_1.ps1                                  UTF-8-SIG
sample_2.txt                                  SHIFT_JIS
sample_3.ps1                                  utf-8
sample_4.txt                                  utf-8
```
